# Class card animal photos

Drop wildlife photos here to give class cards real imagery. Each file must be
named **exactly** `<id>.jpg` (lowercase). Any missing photo falls back to a
themed gradient + animal emoji automatically, so you can add them gradually.

- Landscape photos work best (cards crop to a wide banner, ~2:1).
- Use images you own or that are licensed for use (e.g. your own field photos,
  or CC0 / royalty-free stock). Keep each under ~500 KB for fast loading.
- JPG is expected. If you must use PNG/WEBP, change the extension in
  `src/data/classAnimals.ts` (the `image:` path).
